# captain-jack-sparrow--ai-agent
# 🏴‍☠️ Captain Jack Sparrow AI Assistant

A desktop AI assistant built with Python inspired by JARVIS-style systems.

Captain Jack Sparrow can:

- Open applications
- Open websites
- Listen to voice commands
- Detect wake words
- Control system volume
- Take screenshots
- Shutdown / Restart / Lock PC
- Speak responses using AI voice
- Run with animated futuristic UI

---

# 🚀 Features

## 🎤 Voice Assistant
Supports wake words like:

- "Captain"
- "Jack"
- "Captain Jack Sparrow"

Example commands:

```bash
jack open youtube
jack open chrome
jack volume up
jack screenshot
```

# 📦 Installation

## 1. Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/captain-jack-sparrow-ai.git
```

---

## 2. Open Project

```bash
cd captain-jack-sparrow-ai
```

---

## 3. Create Virtual Environment

```bash
python -m venv venv
```

Activate environment:

### Windows

```bash
venv\Scripts\activate
```

---

## 4. Install Dependencies

```bash
pip install speechrecognition pyttsx3 pyautogui pillow pyaudio
```

---

## 5. Run Project

```bash
python jack_agent.py
```
